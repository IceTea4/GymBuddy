## Project Overview

GymBuddy is a mobile application designed for fitness enthusiasts who struggle with workout planning and tracking. The app provides:
- **Predefined training plans** with structured workouts.
- **Custom workout creation**, allowing users to build personalized routines.
- **Exercise guides** with descriptions and proper execution techniques.
- **A workout calendar** for scheduling and tracking progress.
- **A partner-matching system** to find gym buddies based on time and location.

This app aims to enhance motivation and discipline by providing an easy-to-use and engaging platform for gym-goers.

## Architecture

### System Overview:
- **Frontend:** Android UI (Jetpack Compose & XML)
- **Backend:** Firebase (Planned)
- **Authentication:** Google Sign-In (Planned)
- **Data Storage:** Cloud Firestore (Planned)

### Basic Workflow:
1. User logs in and accesses the home page.
2. Selects a **predefined plan** or **creates a custom plan**.
3. Uses the **calendar** to schedule workouts.
4. Can search for a **workout partner**.
5. Progress and plans are **saved** for future use.
